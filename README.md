# Guess the Word Game

Guess the Word is a simple browser-based game where players try to guess a hidden word using hints. The player must guess the correct word within a limited number of attempts.

This project was built using a single HTML file that includes HTML, CSS, and JavaScript. It helped me practice basic game logic, DOM manipulation, and interactive web development.

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Play

1. Read the hint provided.
2. Guess the correct word.
3. Try to find the word using hints
